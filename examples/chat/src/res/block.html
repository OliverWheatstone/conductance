Blocklambda

SJS adds a "blocklambda" syntax, which is similar to blocks in the ruby language. Blocklambdas are appended to the existing arguments in a function call, so: 

fn(arg) { |x, y| /\* body \*/ }

Is similar to:

fn(arg, function(x, y) { /\* body \*/ })
